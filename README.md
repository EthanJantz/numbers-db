# numbers-db

Numbers are mysterious. We know so little about them! This CLI tool allows the user to learn more about numbers.

# Getting Started

Create a .env file in the root folder with the appropriate values. 

```
uv sync
```

## Example Usage

```
# Ask about 8431
uv run main.py 8431

# Ask about 8431 using the power of the web
uv run main.py 8431 -e=web
```
